# Edge Case Rulebook

> 邊界情況處理規則

---

## 📋 待定義內容

### 活動相關

- [ ] 活動開始後有人報名怎麼辦？
- [ ] 活動人數滿了但有人取消怎麼辦？
- [ ] Host 臨時取消活動怎麼處理？
- [ ] 活動時間衝突（同一用戶）怎麼處理？

### 票務相關

- [ ] 付款失敗但庫存已鎖定？
- [ ] 退票時超過期限？
- [ ] 部分退票（套票）？

### 聊天相關

- [ ] 違規訊息處理
- [ ] 聊天室封存後有人想發言？

### 訂閱相關

- [ ] 訂閱到期但有進行中活動？
- [ ] 降級時超過新等級限制？
