Party PRD

Versioned Product Requirements Document（V0–V3）

⸻

0. 文件說明
 • 產品名稱：Party
 • 產品類型：活動聚合 × 社交 × 商業化平台
 • 本文件用途：
 • 作為產品總 PRD
 • 指導 roadmap 與版本切割
 • 作為設計 / 開發 / 商務的共用語言
 • 版本原則：
 • V0 驗證
 • V1 變現
 • V2 網路效應
 • V3 平台化

⸻

1. 產品願景與核心假設

1.1 願景

打造一個以「活動」為最小單位，能同時承載：
 • 參與
 • 社交
 • 信任
 • 商業轉化

的 活動中樞型平台。

1.2 核心假設
 • 活動本身即是最強的社交入口
 • 活動前後的互動能顯著提高留存
 • Host 是平台最重要的供給端
 • 工具與曝光 > 純流量，是 Host 付費的真正動機

⸻

2. 目標用戶（全版本通用）

2.1 主要角色
 • Attendee（參與者）
 • Host（活動發起者）
 • Club Owner（社群經營者）
 • Vendor / Collaborator（協作者）

2.2 關鍵行為
 • 找活動
 • 參加活動
 • 活動內互動
 • 重複參與
 • 發起活動

⸻

V0 — MVP（驗證期）

3. V0 版本目標

驗證「活動 × 即時互動 × 轉化」是否成立

成功指標：
 • 活動能被成功建立並報名
 • 用戶願意在活動中互動
 • Host 願意重複使用

⸻

4. V0 功能需求（Functional Requirements）

4.1 活動核心
 • 建立 / 編輯 / 發布活動
 • 活動瀏覽與報名
 • 活動舉辦形式（最小集）：
 • 滿人即行
 • 計時模式

4.2 票務（基礎）
 • 單票
 • 預購 / 早鳥
 • 套票（不含複雜條件）

4.3 活動互動
 • 活動聊天室（文字）
 • 僅 Host 可開啟
 • 活動結束後自動封存

4.4 簽到
 • QR Code 簽到
 • Check-in 成功紀錄

4.5 行銷呈現
 • 靜態 Banner
 • 活動倒計時

4.6 數據（BI）
 • Amplitude 基礎事件追蹤
 • view_event
 • join_event
 • open_chat
 • check_in

⸻

5. V0 非目標（Out of Scope）
 • 評論 / 星級
 • 身份折扣
 • Club
 • 外部平台聯動

⸻

V1 — 商業化（付費驗證期）

6. V1 版本目標

讓 Host 願意為「工具 × 互動 × 曝光」付費

成功指標：
 • 有實際訂閱或單次付費
 • 聊天 / 討論使用率提升
 • 活動完成率提升

⸻

7. V1 新增功能需求

7.1 評價與信任系統
 • 活動後星級互評
 • 回評論
 • 檢舉（人工處理）
 • Host 公開評分頁

7.2 聊天室加值
 • 聊天室延長（30 天）
 • 討論串（Thread）
 • 單次解鎖：
 • NTD 30 / 活動 / 30 天

7.3 訂閱制（初版）
 • Free / Plus / Pro
 • 差異點：
 • 活動數量
 • 聊天室權限
 • 數據可視程度

7.4 促銷 Banner
 • 秒殺樣式
 • 強化倒計時視覺

7.5 簽到 × 成就
 • POAP（關鍵活動）
 • 活動完成徽章

⸻

V2 — 網路效應（社群期）

8. V2 版本目標

讓活動不再是一次性，而是形成社群與關係網

成功指標：
 • 重複參與率
 • Club 活躍度
 • 用戶互相關注

⸻

9. V2 新增功能需求

9.1 Club 系統
 • Club 建立與管理
 • Club 專屬聊天室
 • Club 專屬活動
 • Club 會員折扣

9.2 用戶標籤系統
 • 行為標籤（參與 / 發起 / 評論）
 • 活動偏好標籤
 • 問卷式標籤（活動後）
 • 標籤影響：
 • 推薦排序
 • 折扣資格

9.3 活動舉辦形式（完整）
 • 接龍模式
 • 條件模式
 • 滿額即行

9.4 轉發與裂變
 • 分享海報生成
 • 專屬連結
 • QR Code
 • 轉發路徑數據（金字塔）

9.5 地區折扣
 • 城市 / 區域限定優惠

⸻

V3 — 平台化（生態期）

10. V3 版本目標

從活動工具，升級為活動商業中樞

成功指標：
 • 外部合作帶來收入
 • Vendor / 服務方進駐
 • Host 商業化深化

⸻

11. V3 新增功能需求

11.1 外部平台聯動
 • 店家資訊（store.com.tw）
 • 點評 / 套票 / 排行
 • 課程 / 講師平台（school.com.tw）
 • 行銷服務（hyphen / comma）
 • SEO / GEO / AEO（search.com.tw）
 • 客製禮品（sortir）

11.2 商業模組
 • Vendor 曝光付費
 • Host 服務市場
 • 數據報告 / 代營運

⸻

12. 版本總覽表

版本 核心目標 關鍵能力
V0 驗證 活動 + 聊天
V1 變現 評價 + 加值
V2 留存 Club + 標籤
V3 擴張 聯動 + 生態
