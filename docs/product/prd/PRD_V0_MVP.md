## 產品概述

Party Aggregator 是一個活動聚合平台，讓用戶可以發現、報名和管理各類活動。

### 目標用戶

1. **參與者** — 尋找和報名活動
2. **Host** — 創建和管理活動
3. **KOL** — 推廣活動並獲得分潤
4. **Vendor** — 提供活動服務

### 核心價值

- 簡化活動發現流程
- 一鍵報名
- 社交化活動體驗

## 用戶故事

### US-001: 一鍵報名

**作為** 已登入用戶
**我想要** 一鍵報名活動
**以便於** 快速參與感興趣的活動

**AC:**

- [x] 點擊按鈕立即報名
- [x] 報名成功顯示簽到碼
- [ ] 未登入用戶彈出登入頁（PAR-32）
- [ ] 登入後自動跳轉回原頁面（PAR-33）

### US-002: 會員等級限制

**作為** Host
**我想要** 了解我的活動限制
**以便於** 決定是否升級

**AC:**

- [ ] Free 用戶最多 1 場活動
- [ ] Plus 用戶最多 5 場活動
- [ ] 超限時顯示升級提示
