# 🗺️ 簡化版地點選擇器

## 當前版本：不需要 Google Maps API

為了讓您能立即使用地點選擇功能，我們提供了簡化版本，**無需任何 API 配置**！

## ✅ 現有功能

### 1. 快速選擇熱門地點

- 8 個台北熱門地點預設
- 支援搜索篩選
- 自動填入地址和座標

**預設地點**：

- 台北101
- 西門町
- 東區
- 師大夜市
- 信義商圈
- 中山站
- 士林夜市
- 台大校園

### 2. 使用當前位置

- 點擊按鈕自動獲取 GPS 座標
- 無需任何 API

### 3. 手動輸入

- 完全自由輸入地點名稱和地址
- 適合任何地點

## 📝 使用方式

1. **快速選擇**：
   - 點擊或搜索預設地點
   - 自動填入完整資訊

2. **當前位置**：
   - 點擊「使用當前位置」按鈕
   - 瀏覽器請求位置權限後自動填入

3. **手動輸入**：
   - 在下方欄位手動輸入
   - 地點名稱 + 詳細地址

## 🎯 現在就可以測試

```
http://localhost:3000/host/edit
```

滾動到「活動地點」區塊，您將看到簡化版選擇器，**無需任何設置即可使用**。
